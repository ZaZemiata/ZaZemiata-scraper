# ZaZemiata

## Installation - SQLite

1. git clone 
2. cd ZaZemiata
3. npm install
4. Set up `.env` file with the following content:
```
DATABASE_URL="<Here goes your postgres connection string!>"
```
5. npm run prisma:generate
6. npm run prisma:migrate
7. npm run dev


## Studio
`npm run prisma:studio`
